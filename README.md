# Download 
1. download the project
2. unzip the project 
3. composer install also hit this commond on the directory of this project
4. make a file .env and copy all the text of .env-example and paste in .env file
5. hit this commond "php artisan genrate:key"
6. make a database name lv_customer and config in .env file
7. hit this commond "php artisan migrate"
8. hit this commond and run the application "php artisan serve"
